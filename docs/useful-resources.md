# Useful Resources

## Overview
This section provides a comprehensive collection of resources related to digital preservation and web archiving at ICAEW. These resources are organized by platform and category for easy reference.

## Platform-Specific Resources

### Preservica
- **[Preservica - Universal Access](https://icaew.access.preservica.com)**  
  Primary access point for users

- **[Preservica - Universal Access - WordPress Admin](https://icaew.access.preservica.com/wp-login.php)**  
  WordPress administration interface

- **[Preservica - Admin](https://eu.preservica.com/sso/authentication/login.html)**  
  Administrative interface

- **[Preservica Service Desk](https://preservica.freshdesk.com/support/login)**  
  Technical support and issue resolution

- **[Preservica Community Hub](https://community.preservica.com/)**  
  User community and knowledge base

- **[Preservica User Group](https://usergroup.preservica.com)**  
  Legacy community forum

### Archive-It
- **[Archive-It - Public Collections](https://archive-it.org/organizations/1613)**

- **[Archive-It - Admin](https://partner.archive-it.org/)**  
  Administrative interface

- **[Archive-It - Help Centre and Training](https://support.archive-it.org/hc/en-us/articles/208117886-Live-Archive-It-Training-Sessions-and-Webinars)**
