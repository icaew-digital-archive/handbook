# Overview

Preservica houses both **public** and **restricted** collections. 

- **Public collections** contain assets that are fully accessible to the public.
- **Restricted collections** contain assets that require logging in to view.


To access the restricted collections, please ensure you are logged in at the access portal: [https://icaew.access.preservica.com/](https://icaew.access.preservica.com/).

A username and password can be obtained upon request from the Digital Archive Manager or Digital Archivist.
